Requirements/Dependencies:

- Latest versions of JDK 
  (http://www.oracle.com/technetwork/java/javase/downloads/index.html)
- Latest Version of Java3D 
  (http://www.oracle.com/technetwork/articles/javase/index-jsp-138252.html)
- MATLAB)2014b + SibBiology Toolkit 
  (previous versions may work but will not support some of the models due limitated SBML support in earlier version of MATLAB)
- LibSBML for Java and Matlab Bindings
  (http://sbml.org/Software/libSBML)

- MATLAB_scripts folder must be located in the SiViT's main directory.
- Java3D and Matlab installation paths should be detected automatically.
